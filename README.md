# OPD Queue Management System

## Overview
This project is a scalable full-stack healthcare application designed to streamline patient flow and reduce waiting time in clinics through efficient OPD queue management.

- Designed using a modular backend architecture with Node.js and Express.js to ensure maintainability and scalability  
- Developed RESTful APIs with focus on backend API optimization and efficient request handling  
- Implemented secure authentication using JWT (JSON Web Token)  
- Structured database using MongoDB for efficient storage and retrieval of patient and queue data  
- Built a responsive frontend using React.js with smooth API integration and state management  
- Focused on performance optimization, clean code practices, and real-world usability  

## Features
- Patient Registration & Login  
- Appointment Booking  
- Real-time Queue Tracking  
- Admin Dashboard for managing patients  

## Tech Stack
- MongoDB (Database)  
- Express.js (Backend Framework)  
- React.js (Frontend)  
- Node.js (Runtime Environment)  
- JWT (JSON Web Token) Authentication  

## Project Structure
- /frontend → React-based UI  
- /backend → Node.js + Express APIs
  
## How to Run
## Backend
cd backend  
npm install  
npm start  

## Frontend
cd frontend  
npm install  
npm start
